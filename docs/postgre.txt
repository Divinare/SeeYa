Connects to database:
\connect devdb

Drops a database:
DROP DATABASE devdb;

Creates a database:
CREATE DATABASE devdb;

List all tables:
\d
\d+
\l+

Select all events:
select * from "Events";





DATABASE management:

We do this by using dump files, read documentation: http://www.postgresql.org/docs/9.4/static/backup-dump.html

Create a dumpfile of a dabatase, outfile = some output filename:
pg_dump devdb > outfile

To import into a existing EMPTY database:
psql devdb < infile
